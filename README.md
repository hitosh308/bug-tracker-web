# 目的
システム開発におけるテスト工程で発生したバグ情報を一元管理するツール

# 主な機能
* バグ登録
  * 管理項目
    * 件名
    * 発生システム
    * 起票者
    * 起票日時
    * 発生プロジェクト
    * 発生テスト工程
    * 発生テスト番号
    * 発生事象
    * 発生手順
    * 発生日
    * 発生環境
    * 発生個所
    * 再現性（あり／なし）
    * 影響度（高／中／低）
* バグトラッキング
  * 管理項目
    * 原因
    * 担当者
    * 対応日時
    * 対応内容
    * バグ分類
    * バグ混入工程
    * バグ混入ベンダー
    * ステータス（未対応／対応中／対応完了）
* バグ傾向可視化
  * バグ分類傾向
  * バグ混入工程傾向
  * バグ混入ベンダー傾向
* バグに関連するマスタ情報管理
  * システムマスタ
    * システムコード／名称
  * プロジェクトマスタ
    * プロジェクト番号
    * リリース予定年月
  * ベンダーマスタ
    * ベンダーコード／名称

# 画面一覧

    * バグ登録画面
    * バグトラッキング画面
  * 可視化
    * バグ分類傾向画面
    * バグ混入工程傾向画面
    * バグ混入ベンダー傾向画面
  * マスタ管理
    * システムマスタ
      * システムマスタ一覧画面
      * システムマスタ登録画面
    * プロジェクトマスタ
      * プロジェクトマスタ一覧画面
      * プロジェクトマスタ登録画面
    * ベンダーマスタ
      * ベンダーマスタ一覧画面
      * ベンダーマスタ登録画面
